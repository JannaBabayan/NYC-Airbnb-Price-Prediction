## Վերջնական Արդյունքներ և Մետրիկաներ
Մոդելների ճշգրտությունը և կանխատեսման որակը գնահատվել են ռեգրեսիայի հիմնարար չափանիշներով՝ **MAE**, **MSE** և **RMSE**:

### Բացահայտումներ.
* **Գծային մոդելների ձախողումը.** Linear Regression-ը և Lasso-ն ցույց տվեցին ցածր արդյունավետություն, ինչն ապացուցում է, որ անշարժ գույքի շուկայի գործոններն ունեն բարդ, ոչ գծային փոխազդեցություններ:
* **KNN-ի սահմանափակումը.** Քանի որ խնդիրը հանդիսանում է ռեգրեսիա, KNN-ի որակը գնահատվել է սխալների չափման մետրիկաներով (MAE, MSE, RMSE), որտեղ այն ցույց տվեց միջին արդյունք՝ տուժելով չափողականության անեծքից (*Curse of Dimensionality*):
* **Լավագույն Մոդել (Random Forest).** Տվյալների $1000-ի սահմաններում մաքրումից (Outlier Handling) հետո, **Random Forest Regressor**-ը գերազանցեց բոլոր մոդելներին: Շնորհիվ իր *Bagging* կառուցվածքի (բազմաթիվ անկախ ծառերի միջինացման), այն իդեալականորեն ընդհանրացրեց «նորմալ» գնային տիրույթի օրինաչափությունները՝ ապահովելով նվազագույն սխալներ և կայուն $R^2$ գործակից:
